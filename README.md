# Журнал изменений

## v0.6.0 (2025-11-20)

### Сделано

- Авторизация и доступ по ролям.
- Сохранение диалога в БД для будущего накопления знаний.
- Прикрепление файлов к задаче.
- Скачивание брифа в PDF.
- Сообщения об ошибках, а не падение приложения.
- Создание и редактирование категории услуги по месту.

### Исправлено

- Ошибка подключения к БД при первом подключении.
- Не передавался промт для ИИ.

## v0.5.0 (2025-11-15)

### Сделано

- Связка демо-промта и демо задачи, чтобы можно было пробовать редактировать промт и проверять его в чате.
- Чат теперь использует демо-промт и его модель ИИ.
- Сервер ИИ научился возвращать ошибки, чтобы не было молчания на фронте.
- Доработан системный промт для совмещения с промтом пользователя.
- ИИ теперь понимает, когда пользователь отредактировал документ.

### Исправлено

- Не верное сообщение при редактировании промта.

## v0.4.0 (2025-11-06)

Этот релиз посвящен проверке, что возможно сделать удобное редатирование документа совместно с ИИ.

### Сделано

- Страница задач. Одна демо задача для обкатки.
- Страница задачи. Просмотр и переход к брифу.
- Страница брифа.
- Сервер ИИ с минимальным промтом и дефолтной моделью ИИ.
- Совместное редактирование документа с ИИ.
- Много улучшалок UI/UX.

## v0.3.0 (2025-10-30)

### Сделано

- Страница промтов.
- Создание и редактирование промтов.
- Архивация и удаление промтов.
- Просмотр архива и восстановление промтов.
- Группировка промтов по услугам.
- Управление статусом промтов.
- Полнотекстовый редактор с автосохранением изменений.

### Доработано

- Новая верстка услуг - теперь они сгруппированы по категориям.

### Исправлено

- При первом запуске сервера, приложение выдавало ошибку подключения к БД.
